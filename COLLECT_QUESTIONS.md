# Инструкция по сбору вопросов для тестов

## Важно

Текущие JSON файлы содержат только тестовые данные (по 2 вопроса). Необходимо собрать полные наборы вопросов согласно официальной спецификации.

⚠️ **ВАЖНО:** Согласно ТЗ, нужно собирать **ВСЕ 4 типа вопросов**, включая вопросы с аудио и картинками. Медиафайлы нужно будет скачать отдельно.

## Источник вопросов

Официальный сайт: https://inostr-exam.fipi.ru/xmodules/qprint/index.php

## Процесс сбора

1. Перейдите на сайт https://inostr-exam.fipi.ru/xmodules/qprint/index.php
2. Выберите нужный уровень теста (1, 2 или 3)
3. Слева будут отображаться все вопросы теста
4. Для каждого вопроса:
   - Перейдите в вопрос
   - Вы увидите несколько вариантов этого вопроса
   - Выберите один вариант (на ваше усмотрение)
   - Скопируйте данные вопроса

## Структура данных вопроса

### 1. Текстовый вопрос с текстовыми вариантами (text-choice)

```json
{
  "id": "q1",
  "type": "text-choice",
  "text": "Текст вопроса",
  "section": "Лексика-Грамматика",
  "correctAnswer": "opt1",
  "options": [
    { "id": "opt1", "text": "Вариант 1" },
    { "id": "opt2", "text": "Вариант 2" },
    { "id": "opt3", "text": "Вариант 3" }
  ]
}
```

### 2. Текстовый вопрос с аудиозаписью (text-audio-choice)

```json
{
  "id": "q2",
  "type": "text-audio-choice",
  "text": "Текст вопроса",
  "section": "Аудирование",
  "correctAnswer": "optA",
  "audioUrl": "/audio/question-2.mp3",
  "options": [
    { "id": "optA", "text": "Вариант A" },
    { "id": "optB", "text": "Вариант B" }
  ]
}
```

**Важно**: 
- ✅ **ИСПОЛЬЗОВАТЬ** - согласно ТЗ этот тип вопроса обязателен
- Аудиофайлы нужно скачать с официального сайта и разместить в папке `public/audio/`
- Поддерживаемые форматы: `.mp3`, `.wav`, `.ogg` (рекомендуется `.mp3`)
- Путь в JSON должен начинаться с `/audio/`

### 3. Текстовый вопрос с таблицей (text-table-input)

```json
{
  "id": "q3",
  "type": "text-table-input",
  "text": "Текст вопроса",
  "section": "Чтение",
  "correctAnswer": "Правильный ответ",
  "tableData": {
    "Заголовок 1": ["Значение 1"],
    "Заголовок 2": [""]
  }
}
```

### 4. Текстовый вопрос с картинками (text-image-choice)

```json
{
  "id": "q4",
  "type": "text-image-choice",
  "text": "Текст вопроса",
  "section": "Лексика-Грамматика",
  "correctAnswer": "img1",
  "options": [
    { "id": "img1", "imageUrl": "/images/question-4-1.png", "alt": "Описание 1" },
    { "id": "img2", "imageUrl": "/images/question-4-2.png", "alt": "Описание 2" }
  ]
}
```

**Важно**: 
- ✅ **ИСПОЛЬЗОВАТЬ** - согласно ТЗ этот тип вопроса обязателен
- Изображения нужно скачать с официального сайта и разместить в папке `public/images/`
- Поддерживаемые форматы: `.png`, `.jpg`, `.jpeg`, `.webp`
- Путь в JSON должен начинаться с `/images/`
- При ошибке загрузки будет показан placeholder

Все тесты содержат следующие разделы:

5. **Говорение** (Speaking)

## Проверка после сбора

После заполнения JSON файлов проверьте:
1. Все вопросы имеют уникальные ID
2. Все типы вопросов соответствуют одному из 4 типов
3. Все разделы указаны правильно
4. Правильные ответы указаны корректно
5. Аудиофайлы и изображения загружены в соответствующие папки
